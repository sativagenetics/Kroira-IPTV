# KROIRA IPTV 2.0.1 Release Notes

## Scope

This hotfix focuses only on stability, responsiveness, first-run onboarding, source error states, and local diagnostics. It does not add content, providers, licensing, ads, subscriptions, or monetization.

## Release Notes

Improved stability and responsiveness, reduced freezes during source import and playback, improved first-run experience, clearer source error messages, and better handling of large playlists and EPG refreshes.

## Version

- Public release version: `2.0.1`
- Package manifest identity version: `2.0.1.0`
- App assembly and file metadata: `2.0.1.0`

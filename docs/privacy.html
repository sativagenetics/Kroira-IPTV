KROIRA IPTV Privacy Policy

Last updated: April 2026

KROIRA IPTV is a Windows media player and source manager for user-provided M3U playlists, Xtream providers, and Stalker portals. KROIRA does not provide channels, streams, playlists, subscriptions, credentials, or media content.

What the app uses

The app can connect to user-provided source endpoints, XMLTV guide endpoints, poster/artwork endpoints, metadata provider endpoints, and optional local routing or companion endpoints configured by the user.

Local data storage

KROIRA stores app-related data locally on the device, including source configuration, guide settings, cached guide data, favorites, playback progress, continue-watching data, source health diagnostics, protected credential records, and local preferences.

Source credentials

Source credentials are used only to connect to sources that the user configures. KROIRA uses protected credential copies with Windows DPAPI CurrentUser scope when available. Some plaintext source fields may remain for backward compatibility with existing import and parser flows; they are not removed destructively in this release.

Diagnostics and logs

Sanitized diagnostics and exported source reports are designed to mask playlist URLs, usernames, passwords, tokens, keys, MAC-like values, and loose secrets before display or export. Users should still review exported diagnostics before sharing them because provider names and non-secret configuration may remain.

Telemetry

KROIRA does not bundle app telemetry, advertising analytics, or a vendor-operated media URL collection service in this release.

Metadata providers

When metadata enrichment is used, KROIRA may contact metadata or artwork providers such as TMDb, plus image URLs supplied by user-configured sources, to improve titles, posters, and backdrops. KROIRA does not send source passwords or provider tokens to TMDb.

User responsibility

KROIRA is not a content provider. Users are responsible for adding only authorized sources and for understanding what information their configured sources expose. KROIRA does not bypass DRM, paywalls, authentication, or access controls.

Network access

Network access is used to connect to user-provided IPTV sources, XMLTV guide data, metadata/artwork endpoints, and optional local companion or proxy services that the user configures.

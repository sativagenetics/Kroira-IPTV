KROIRA IPTV Privacy Policy

Last updated: April 2026

KROIRA IPTV is a Windows desktop IPTV player for user-provided sources. It supports user-provided M3U playlists, Xtream providers, and Stalker portals. KROIRA does not provide channels, playlists, credentials, subscriptions, or hosted content.

What the app uses

The app can connect to user-provided source endpoints, guide endpoints, poster artwork endpoints, and optional local routing or companion endpoints configured by the user.

Local data

KROIRA stores app-related data locally on the device, including source configuration, guide settings, favorites, playback progress, continue-watching data, and local preferences.

Personal data

KROIRA is not a content provider, does not sell user data to advertisers, and does not operate telemetry servers for your media URLs or provider credentials. Users are responsible for the sources they add and the information those sources expose.

Network access

Network access is used to connect to user-provided IPTV sources, XMLTV guide data, metadata/artwork endpoints, and optional local companion or proxy services that the user configures.

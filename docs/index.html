KROIRA IPTV

Windows IPTV media player and source manager

KROIRA IPTV is a packaged WinUI 3 desktop app for Windows. It supports user-provided M3U playlists, Xtream providers, and Stalker portals, with embedded playback, guide controls, favorites, source diagnostics, global search, and continue-watching support.

KROIRA does not provide channels, streams, playlists, subscriptions, credentials, or media content. Users are responsible for adding only authorized sources. KROIRA does not bypass DRM, paywalls, authentication, or access controls.

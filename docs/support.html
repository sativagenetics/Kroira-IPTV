KROIRA IPTV Support

KROIRA IPTV is a player-only application for user-provided IPTV sources. It does not provide channels, playlists, subscriptions, or credentials.

Need help?

Use the GitHub issue templates for reproducible bugs and feature requests. For private or sensitive support matters, email batuhandemirbilek7@gmail.com.

GitHub repository

https://github.com/sativagenetics/KroiraIPTV

Include with your report

If possible, mention the source type (M3U, Xtream, or Stalker), the affected area (Live TV, movies, series, guide data, or playback), and the latest sync or health status shown in Sources.

Do not include

Do not send passwords, playlist URLs, MAC addresses, or other private provider credentials in public bug reports.

# Support

## What KROIRA Supports

KROIRA IPTV is a Windows desktop player for user-provided IPTV sources.

Support requests are appropriate for:

- reproducible application bugs
- questions about current app behavior
- build or contributor setup problems
- documentation gaps

Support requests are not appropriate for:

- asking for playlists, channels, or credentials
- provider account recovery
- copyright disputes related to third-party sources
- support for third-party services that KROIRA does not operate

## Where to Ask for Help

## Before Filing an Issue

Include the minimum context needed to reproduce the problem:

- source type: M3U, Xtream, or Stalker
- affected area: sources, live TV, movies, series, guide data, playback, favorites, continue watching, or settings
- app build/version if known
- Windows version if relevant
- steps to reproduce
- expected result
- actual result

Do not include:

- personal playlist URLs
- usernames, passwords, MAC addresses, or tokens
- private provider screenshots that expose account data
